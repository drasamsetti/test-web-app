Login Screen


Once the user opens the site, he is presented with the following screen. Registered email id and the password combination can be used to login. For New User Registration, click on the 'Register' link visible just below the area where email id and password are asked for login.


Once successfully logged in, user lands at Account Dashboard. This is the account to which user is primarily associated.



Forgot password

In case the user forgets the password, then the user can get the password mailed to his/her email Id by entering the his e-mail id in the “ENTER USER ID” TAB and click on the  SVBMIT button. The password will be emailed to your CSC e-mail id.
