New User Registration


Clicking on 'Register' Link on the Login Screen leads to the following screen. Here, a new user account (User) can be created. New user has to be associated to one of the Accounts and one of the categories defined by the tool administrator. The user thus created will be subject to approval of the account administrator and he will assign a specific role to this user which is by default a simple user.


to be precise, the following process need to be followed to get registered into the tool:

  1. First click on the “REGISTER” tab in the login page.
  2. Then it directs you to the following screen. Fill the details as:

User id (Email Address):  your csc email id.

Full Name: your name.

Password: choose accordingly to your wish.

Re-enter password: reenter your password.

Enter manager’s name: enter your manager name

Enter manager’s Email address: Enter your manager mail id.

Account: Select an account in the dropdown which you belongs to.


             Category: Select a category in the dropdown which you belongs to.



  1. Click on submit button

Note that the user registration completes only after his/her account administrator approves.
