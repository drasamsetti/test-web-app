Features

Features of Idea Generation Tool

  • Capturing all the information about an idea - Growing from a "post it" to a full-fledged idea evaluation report
  • Evaluation of ideas by experts
  • Out-of-the-box - No customization required.
  • Idea pool - Knowledge sharing across organizational boundaries
  • Reporting - Compiling up-to-date idea data into Excel reports
  • Dashboards – Rich graphical dashboard depicting day to day statistics
  • Security – Account level security, themes can be restricted to individual accounts
  • Web-based - All features are accessible using any browser.
  • No local installation required - Software and updates need only be installed on the server, not on every user's computer
